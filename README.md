# QuizNova AI

QuizNova AI is a modern quiz application built with Next.js, MongoDB, and Google's Generative AI. It allows users to create, take, and track their progress on quizzes with AI-generated questions and explanations.

## Features

- **User Authentication**: Secure login and registration system
- **AI-Generated Quizzes**: Create quizzes with questions generated by Google's Generative AI
- **Difficulty Levels**: Choose from easy, medium, and hard difficulty levels
- **Progress Tracking**: Track your quiz performance over time
- **Responsive Design**: Works on desktop and mobile devices
- **PDF Upload**: Upload PDFs to generate quizzes from your content

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT with HTTP-only cookies
- **AI Integration**: Google Generative AI
- **Email**: Nodemailer for password reset
- **Charts**: Recharts for progress visualization

## Prerequisites

- Node.js 18+ and npm
- MongoDB Atlas account (or local MongoDB)
- Google AI API key
- Email account for password reset functionality

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/quiznova-ai.git
cd quiznova-ai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up environment variables

Copy the `.env.example` file to `.env` and fill in your credentials:

```bash
cp .env.example .env
```

Edit the `.env` file with your actual credentials:

```
# MongoDB Connection
MONGODB_URI=mongodb+srv://your-mongodb-uri

# JWT Authentication
JWT_SECRET=your-secure-jwt-secret-key

# Google AI API
GOOGLE_AI_API_KEY=your-google-ai-api-key

# Email Configuration (for password reset)
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-email-app-password
EMAIL_FROM=your-email@gmail.com

# Next.js Configuration
NEXT_PUBLIC_API_URL=http://localhost:3000/api
```

### 4. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Deployment

### Deploying to Vercel

1. Push your code to a GitHub repository
2. Import the repository to Vercel
3. Set up the environment variables in the Vercel dashboard
4. Deploy!

## Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `MONGODB_URI` | MongoDB connection string | Yes |
| `JWT_SECRET` | Secret key for JWT token generation | Yes |
| `GOOGLE_AI_API_KEY` | API key for Google's Generative AI | Yes |
| `EMAIL_USER` | Email address for sending password reset emails | Yes |
| `EMAIL_PASS` | Email password or app password | Yes |
| `EMAIL_FROM` | Sender email address | Yes |
| `NEXT_PUBLIC_API_URL` | Base URL for API requests | Yes |

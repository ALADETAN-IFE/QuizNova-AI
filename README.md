# QuizNova AI

QuizNova AI is a modern quiz application built with Next.js, MongoDB, and Google's Generative AI. It allows users to create, take, and track their progress on quizzes with AI-generated questions and explanations.

## Features

- **User Authentication**: Secure login and registration system with Google OAuth and email/password
- **AI-Generated Quizzes**: Create quizzes with questions generated by Google's Generative AI
- **Difficulty Levels**: Choose from easy, medium, and hard difficulty levels
- **Progress Tracking**: Track your quiz performance over time with detailed analytics
- **Responsive Design**: Works on desktop and mobile devices
- **PDF Upload**: Upload PDFs to generate quizzes from your content
- **Professional Email Notifications**: Beautiful, responsive email templates for welcome and password reset

## Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: MongoDB with Mongoose
- **Authentication**: NextAuth.js with JWT and HTTP-only cookies
- **AI Integration**: Google Generative AI
- **Email**: Nodemailer with modern HTML templates
- **Charts**: Recharts for progress visualization
- **State Management**: Zustand for global state

## Prerequisites

- Node.js 18+ and npm
- MongoDB Atlas account (or local MongoDB)
- Google AI API key
- Gmail account for email functionality

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/quiznova-ai.git
cd quiznova-ai
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up environment variables

Copy the `.env.example` file to `.env.local` and fill in your credentials:

```bash
cp .env.example .env.local
```

Edit the `.env.local` file with your actual credentials:

```
# MongoDB Connection
MONGODB_URI=mongodb+srv://your-mongodb-uri

# NextAuth Configuration
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-nextauth-secret

# Google OAuth
GOOGLE_CLIENT_ID=your-google-client-id
GOOGLE_CLIENT_SECRET=your-google-client-secret

# Google AI API
NEXT_PUBLIC_GEMINI_API_KEY=your-google-ai-api-key

# Email Configuration
GMAIL_USER=your-gmail@gmail.com
GMAIL_APP_PASSWORD=your-gmail-app-password

# Next.js Configuration
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### 4. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Deployment

### Deploying to Vercel

1. Push your code to a GitHub repository
2. Import the repository to Vercel
3. Set up the environment variables in the Vercel dashboard
4. Deploy!

## Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| `MONGODB_URI` | MongoDB connection string | Yes |
| `JWT_SECRET` | Secret key for JWT token generation | Yes |
| `NEXTAUTH_URL` | Base URL of your application | Yes |
| `NEXTAUTH_SECRET` | Secret key for NextAuth.js | Yes |
| `GOOGLE_CLIENT_ID` | Google OAuth client ID | Yes |
| `GOOGLE_CLIENT_SECRET` | Google OAuth client secret | Yes |
| `NEXT_PUBLIC_GEMINI_API_KEY` | API key for Google's Generative AI | Yes |
| `GMAIL_USER` | Gmail address for sending emails | Yes |
| `GMAIL_APP_PASSWORD` | Gmail app password | Yes |
| `NEXT_PUBLIC_APP_URL` | Base URL for the application | Yes |
